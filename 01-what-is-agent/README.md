# 01-what-is-agent

## 这一章学什么

这一章帮你建立最关键的第一层理解：

`Agent` 到底是什么？

你会看到它和普通 chat 的区别，也会第一次接触最小 `Agent loop`。

## 学完后你会得到什么

完成后，你应该能回答：

- 为什么 `Agent` 不只是聊天
- 为什么 `Tool` 很关键
- 一个最小 `Agent` 为什么会有“输入、决策、行动、观察、回复”这些环节

## 建议阅读顺序

1. `agent-vs-chat.md`
2. `the-minimal-agent-loop.md`

## 本章建议阅读

- Anthropic: [Building Effective AI Agents](https://www.anthropic.com/engineering/building-effective-agents)
  为什么现在读：你刚好在建立第一层认知，这篇会帮你避免把 `Agent` 想得过度神秘或过度复杂。

- Jeff Su: [AI Agents, Clearly Explained](https://www.youtube.com/watch?v=FwOTs4UxQS4)
  为什么现在看：按可获取的摘要来看，这个视频会把 `LLM`、`AI workflow`、`AI agent` 三层关系拆得比较清楚，很适合零基础建立直觉。
  重点看什么：`workflow` 和 `agent` 的区别，以及“能自主决定并迭代”为什么是 `agent` 的关键特征。
  建议程度：补充观看
