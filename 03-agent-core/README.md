# 03-agent-core

## 这一章学什么

这一章开始进入真正的 `Agent` 核心。

你会理解：

- 什么是 `Tool`
- 程序如何做最简单的 `decision`
- 为什么 `observation` 很重要
- 为什么最后还要有 `reply`

## 学完后你会得到什么

完成后，你应该能看懂你的小项目为什么不是“几个零散函数拼起来”，而是一个最小 `Agent`。

## 建议阅读顺序

1. `tools.md`
2. `decision-loop.md`
3. `observation-and-reply.md`
