# Tools

## 什么是 `Tool`

在这个学习项目里，你可以把 `Tool` 理解成：

`Agent` 能调用的一种动作。

比如：

- `get_time`
- `save_note`
- `read_notes`

它们都不是“解释”，而是“动作”。

## 为什么 `Tool` 重要

如果没有 `Tool`，系统大多只能停留在“生成回复”。

有了 `Tool`，`Agent` 才能开始和真实世界发生联系，比如：

- 读取时间
- 保存内容
- 查询文件
- 调用 API

所以你可以把 `Tool` 当成 `Agent` 的“手”和“眼”。

## 在你的第一个项目里，`Tool` 长什么样

在更复杂的系统里，工具可能会有专门格式、参数定义、JSON schema。

但在你的第一个项目里，我们先用最直观的形式：

函数。

比如：

```python
def get_time():
    ...
```

这就足够了。

## 暂时先记住

`Tool` 不是“神秘能力”，而是：

“程序被允许调用的一段明确动作。”
