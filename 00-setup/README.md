# 00-setup

## 这一章学什么

这一章只做一件事：把学习环境准备好。

你会完成：

- 确认 `conda` 可用
- 创建专属的 `conda environment`
- 用 `VS Code` 打开这个学习目录
- 安装 `Python` 和 `Pylance` 扩展
- 运行第一个 `Python` 文件

## 学完后你会得到什么

完成后，你会有一个干净、可重复使用的开发环境。

这很重要，因为后面很多“明明代码没问题却跑不起来”的情况，其实都来自环境没配好。

## 建议阅读顺序

1. `conda-and-vscode.md`
2. `hello.py`

如果你在 `VS Code` 里搜不到 `Python: Select Interpreter`，先不要慌。

这通常不是你操作错了，而是还没安装 `Python` 扩展。这个问题已经在 `conda-and-vscode.md` 里专门补上了。

## 本章建议阅读

这一章暂时不需要额外官方阅读。

你的目标不是“看文档看明白”，而是把环境真的跑通。
