# 07-next-steps

做完第一个小项目后，你已经不是“只听说过 Agent”的阶段了。

接下来你可以沿几个方向继续升级。

## 方向 1：给 Agent 接入 LLM API

你现在的项目是规则驱动的。

下一步可以把“选择工具”这一步逐渐交给模型，让它更灵活。

## 方向 2：增加更多工具

比如：

- 读取本地文件
- 总结一段文本
- 调用天气 API

这会让你更理解 `Tool` 的边界设计。

## 方向 3：加入简单 Memory

你现在的 `notes.txt` 其实已经有一点“外部记忆”的味道了。

后面可以继续做：

- 保存历史对话
- 保存用户偏好
- 在后续任务里引用过去记录

## 方向 4：学习 RAG

当你想让系统基于资料回答问题，而不是只靠模型本身时，就会进入 `RAG`。

## 方向 5：再看框架

到这一步再去看：

- `OpenAI Agents SDK`
- `LangChain`
- `LangGraph`

你会更清楚它们到底在帮你解决什么问题。

## 我的建议

下一步优先顺序可以是：

1. 先把当前项目跑通并改一改
2. 再增加一个新工具
3. 然后考虑接入真实模型
