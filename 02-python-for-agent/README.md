# 02-python-for-agent

## 这一章学什么

这一章不是完整的 `Python` 课程。

它只讲你为了完成第一个 `Agent` 项目真正需要的那部分内容。

## 学完后你会得到什么

完成后，你应该能看懂并写出这类最小代码：

- 定义变量
- 写一个函数
- 用 `if` 做判断
- 从另一个文件 `import` 函数
- 读写本地文件

## 建议阅读顺序

1. `python-basics-for-this-project.md`
2. `files-and-functions.md`

## 本章建议阅读

这一章先不强制配官方阅读。

你的目标是先把“项目必需的 Python”学够，而不是把 Python 全部学完。
